# Проект 4: Место

## Описание проекта
Cервис Mesto: интерактивная страница с личным профилем, куда можно добавлять фотографии, удалять их и ставить лайки.

### Что использовалось?
* HTML и CSS;
* Адаптиваня верстка;
* Создана файловая структура Nested по БЭМ;
* Флексбокс верстка и грид;
* JavaScript
